# Fall

Fall is a simple benchmark app to test Qt Quick animations. It supports Qt versions > 5.12, including Qt 6 (tested up to 6.2.0).

![shot](shot.webp)

# Toradex Apalis iMX6D

- [x] Disable bubbles, then fps increases to 51
